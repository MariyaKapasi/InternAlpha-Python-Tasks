# InternAlpha-Python-Tasks
## ✅ Task 1 – Tic Tac Toe Game

Developed a two-player Tic Tac Toe game in Python using basic programming concepts and game logic. The project includes a 3x3 game board, player turn management, win/tie condition checking, and replay functionality. This project helped strengthen understanding of loops, conditional statements, functions, and list operations in Python.

---

## ✅ Task 2 – Online Banking System (GUI)

Created a GUI-based Online Banking System using Python and Tkinter. The application allows users to perform banking operations such as deposit, withdrawal, and balance enquiry through a user-friendly graphical interface. The project also includes error handling for invalid inputs and insufficient balance, enhancing practical knowledge of GUI development and event handling.

---

## ✅ Task 3 – Snake Game

Built an interactive Snake Game using Python Turtle Graphics. The game features snake movement control using keyboard keys, random food generation, score tracking, and collision detection with walls and the snake’s body. This project improved understanding of game development concepts, animations, event handling, and coordinate-based movement in Python.
